# 接入 DeepSeek(或 GPT / 通义 / 豆包)让小灵变"聪明" · 联调指南

小灵默认用规则引擎(快、离线、省钱)。配一个大模型 API Key 后,复杂开放式对话会走
**智能大脑**(理解上下文 + 用户画像,会思考、会给多种选择)。DeepSeek 国内可直连、便宜,推荐首选。

---

## 一、拿 DeepSeek API Key
1. 打开 https://platform.deepseek.com 注册登录
2. 左侧「API keys」→ 创建一个 → 复制(形如 `sk-xxxxxxxx`)
3. 充值几块钱即可(deepseek-chat 很便宜,老人日常对话一个月几毛到几块)

## 二、给后端配上 Key(设一个环境变量即可)
**本地测试**(电脑上跑后端):
```bash
cd server
export DEEPSEEK_API_KEY=sk-你的key
uvicorn main:app --host 0.0.0.0 --port 8000
```
**云端部署**(Render/Railway/Fly,见 DEPLOY.md):在平台的「Environment / 环境变量」里加
`DEEPSEEK_API_KEY = sk-你的key`,重新部署即可。

> 换别家:改成对应变量即可,一个就够:
> `OPENAI_API_KEY`(GPT) / `DASHSCOPE_API_KEY`(通义) / `ARK_API_KEY`(豆包) / `MOONSHOT_API_KEY`(Kimi)。
> 自定义端点:`XL_LLM_KEY` + `XL_LLM_BASE_URL` + `XL_LLM_MODEL`。

## 三、验证是否启用成功
```bash
curl http://localhost:8000/health
```
返回里 **`"llm": true`** 就是大模型已启用;`false` 说明没读到 Key(检查变量名/是否重启)。

## 四、端到端体验(证明它"会思考")
后端配好 Key 后,手机 App「设置 → 服务器地址」填后端地址;再到「设置 → 个人资料」
填上称呼/爱好/常联系人(如 称呼=王奶奶,爱好=爱听京剧,常联系人=女儿小芳)。

然后对小灵说这些,感受和规则版的差别:
- 「**给小芳打个电话**」→ 大脑知道小芳是女儿,直接拨(规则版可能不认识"小芳")
- 「**我有点闷**」→ 大脑结合"爱听京剧"给你 ①放段京剧 ②聊聊天 ③给女儿打电话 三个选项
- 「**刚才那个再放一遍**」→ 大脑靠会话记忆理解"那个"指上一次播放的内容
- 「**帮我把'我不舒服'翻译成英文**」→ 直接翻译

对比:没配 Key 时这些走规则/兜底,能用但不"懂你";配了 Key 就明显更聪明。

## 五、成本与降级(放心)
- **分层省钱**:防诈/呼救/打电话/导航/翻译等高频+安全指令**仍走规则**(0 成本、离线、快),
  只有开放式对话才调大模型。所以账单很小。
- **超时降级**:大模型 6 秒不回就退回规则/兜底,不会卡住,更不影响呼救/防诈。
- **隐私**:大模型跑在你自己的后端,请求由你的后端转发;不把用户数据交给第三方 App。

## 六、常见问题
- `llm:false`:环境变量没生效 → 确认变量名拼写、配完**重启/重新部署**后端。
- 国内连不上 OpenAI:用 **DeepSeek**(国内直连)或通义/豆包。
- 回复慢:DeepSeek 高峰偶有延迟;小灵已设 6s 超时降级,不会一直等。
- 想更懂用户:让老人在「个人资料」多填爱好和常联系人——大脑会据此更贴心。
